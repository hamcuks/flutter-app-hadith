# Kumpulan Hadith

Merupakan aplikasi yang menyediakan kumpulan hadith dari beberapa perawi. Aplikasi ini dibagun menggunakan Framework Flutter

### Dart Package

- quatable: ^2.0.3
- flutter_riverpod: ^0.14.0+3
- http: ^0.13.3

### Font Icon

- Iconsax by: https://iconsax.io/

### Design
